

==== What is it ? ====

This application is a buil radiator to monitor builds status over a TeamCity continous integration server.


==== Quick Start ====

    1. Unzip the archive
    2. Define JAVA_HOME environement variable or edit start script to link to a JRE 8
    3. Launch the application by script start.sh on Linux or start.bat on Windows
    4. Enjoy !
